# JavaScript To-Do List App

A clean and functional To-Do List built using **HTML, CSS, and Vanilla JavaScript**.

## Features
- Add, delete, and mark tasks as done
- Data persistence using localStorage
- Dark / Light theme toggle
- Responsive layout
- Deployed on GitHub Pages

## Skills Demonstrated
- DOM Manipulation
- State Management
- JavaScript Logic
- Local Storage
- UI/UX Basics
